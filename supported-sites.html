Supported Platforms / 支持平台

NovelPure is designed for chapter-based web novel platforms and works on many structured reading websites.

NovelPure 专为章节结构清晰的小说网站设计，可在多种阅读平台上提供沉浸式阅读体验。

We continuously improve compatibility based on user feedback and website structure updates.

我们会根据用户反馈与网站结构变化持续优化兼容性。

Optimized Platforms / 深度优化平台

  • Royal Road
  • Selected English web novel platforms
  • Selected Chinese reading platforms

These platforms have been specifically tested to provide a stable immersive reading experience.

以上平台经过专项测试，可提供更稳定的阅读模式体验。

Community-Supported Sites / 社区支持

NovelPure may also work on other chapter-based novel websites depending on page structure.

只要页面结构为章节阅读模式，NovelPure 也可能正常运行。

If your favorite site is not supported yet, feel free to submit a request.

如果你常用的网站尚未支持，欢迎提交适配请求。

Compatibility may vary depending on layout or structural updates.

由于网站布局或结构更新，兼容性可能会发生变化。

Request Site Support / 提交站点适配

If NovelPure does not work properly on a chapter page, please let us know.

如果 NovelPure 在某个章节页面未能正常运行，欢迎提交反馈。

Email Support / 邮件反馈 GitHub Issues (Recommended) / GitHub 提交（推荐）

Tip: Please include the page URL and a screenshot for faster troubleshooting.

Safety Notice / 安全说明

NovelPure only enhances the visual presentation of reading pages locally in your browser.

NovelPure 仅在浏览器本地优化阅读页面显示效果。

It does not interact with account systems, payment pages, or non-reading content.

不会作用于账号系统、支付页面或非阅读相关页面。
